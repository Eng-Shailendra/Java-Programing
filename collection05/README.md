# List

**List** is a interface of java collection framework. It is used to store the dataStructure of the same type of Elemnet

## List method

**Add element in list **

- add(), addAll(),

**Access element in list **

- get(), set(), subList(),

**Removeing element **

- remove(), removeAll(), Clear()

**Cheacking and Search Elemnet **

- size(), isEmpty(), contains(), indexOf()

## List has four type

1. ArrayList
2. Vector
3. Stack
4. LinkList

## ArrayList

- ArrayList is a class which implemnted List interface of collection frame work
- it Internally use dymnamic array as the internal implemntation of data Structure which can grow and shrink as per requriremnet
- In array List element stored in continous block of memory
- Default capicaty of arraylist is **10** when capicity is full then new arraylist created internaly with **50%**
- fast In access
- Slow in modification
- it is not synchronusied but if sync is requrired we can synchronised by calling synchronised list of collection class
